Hot-Plug Volumes

Harvester supports adding hot-plug volumes to a running VM.

Currently, KubeVirt only supports disk bus scsi for hot-plug volumes. For more information, see this issue.

Adding Hot-Plug Volumes to a Running VM

The following steps assume that you have a running VM and a ready volume:

  1. Go to the Virtual Machines page.

  2. Find the VM that you want to add a volume to and select ⋮ > Add Volume.

  3. Enter the Name and select the Volume.

  4. Click Apply.
